# Kaspa Address Generator

Rust-приложение для генерации Kaspa mnemonic, приватного/публичного ключа и mainnet адреса.

## Запуск

1. Установите зависимости:
   - kaspa-bip32
   - kaspa-addresses
   - hex

2. Соберите проект:
```
cargo build --release
```

3. Запустите:
```
cargo run --release
```

## Алгоритм
Используется генерация по примеру test1115.rs.
